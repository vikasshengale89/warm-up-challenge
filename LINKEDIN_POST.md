Just shipped an adaptive intelligent learning assistant! 🚀

For the Google Antigravity PromptWars competition, I challenged myself to build "warm-up-challenge" end-to-end. It's a platform that takes complex topics and automatically generates personalized learning modules with interactive quizzes to adapt to your pace.

The best part? It features a stunning Glassmorphism UI and an ambient idle mode with CSS particle animations to keep you focused.

Built entirely with:
⚡ Angular 19 (Signals & Zoneless)
⚡ Node.js + Express backend
⚡ Firebase Hosting
