## Challenge Description

Create an intelligent assistant that helps users learn new concepts effectively. The system should personalize content and adapt to user pace and understanding.

## Functional Requirements

1. **User Goal Setting:** Users can input a specific concept they want to learn and set a target difficulty level.
2. **Concept Breakdown:** The system breaks down the requested concept into 3-5 manageable learning modules.
3. **Interactive Quizzes:** Each module includes a short, interactive quiz to test user understanding before proceeding.
4. **Adaptive Pacing:** If a user fails a quiz, the system provides supplementary material and re-tests them before moving on.
5. **Progress Tracking:** The application visually tracks progress through the modules via a completion dashboard.
6. **Glassmorphism UI:** The entire interface uses a modern Glassmorphism theme with a mesh gradient background (#0F172A and #1E293B), frosted glass effect (12px blur), 1px border with 20% opacity white, and 1rem rounded corners.
7. **Accessibility (A11y):** All UI components are accessible, including alt tags for images, aria-labels for buttons, and WCAG AA compliant color contrast.
8. **Real-time Ambient Idle Mode:** (WOW Factor) The app features a real-time ambient idle mode with CSS particle animations and an event countdown when the user is inactive.
9. **Efficiency & Performance:** The app implements lazy loading for modules and caches previous module data to prevent redundant API calls, keeping load times under 2s.
10. **Problem Statement Alignment:** The application directly addresses the core problem by focusing purely on personalized, adaptive learning without feature creep like social sharing or leaderboards.

## Non-Functional Requirements

- **Performance:** Application must load initial view in < 2 seconds. API responses under 500ms.
- **Security:** Strict XSS prevention, CORS configured for localhost/firebase, and thorough input validation on the backend.
- **Accessibility:** WCAG 2.1 AA compliance (keyboard navigation, screen reader support).
- **Responsive Design:** Mobile-first design that scales perfectly to tablet and desktop resolutions.

## Tech Stack

- **Frontend:** Angular 19 (Standalone, Zoneless, Signals)
- **Backend:** Node.js + Express
- **Deployment:** Firebase Hosting (Spark Plan)
